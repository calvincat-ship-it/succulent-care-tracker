# 寶貝多肉APP

純前端網頁 App（PWA），用來記錄多肉植物（景天科／仙人球／百合科／天門冬科／鳳梨科／番杏科／大戟科／龍舌蘭／夾竹桃科）的購入資訊與日常照護，具備：新增／編輯／刪除植物（軟刪除，保留文字資料供除錯校對）、拍照附加、依分類的「分類卡片」（可一鍵將整個分類標記為已澆水／已施肥）、個別植物卡片各自的澆水／施肥追蹤與提醒、彙整統計（數量／金額，含趨勢圖）、可自訂各分類的澆水上下限天數與施肥上限天數。資料存在手機瀏覽器的 localStorage，照片存在瀏覽器的 IndexedDB，都不需要網路或帳號。

## 分類卡片與個別植物卡片

- **分類卡片**（景天科／仙人球／百合科／天門冬科／鳳梨科／番杏科／大戟科／龍舌蘭／夾竹桃科）會自動顯示該分類底下所有植物中「最新一次」的澆水／施肥日期，並根據設定的天數上下限顯示警示（太久沒澆水／才剛澆水沒多久／太久沒施肥）。按「全部已澆水」或「全部已施肥」會把該分類底下**所有**植物的日期一次更新為今天，適合同分類植物常常一起澆水的情境。
- **個別植物卡片**在「我的多肉」清單中，各自獨立追蹤澆水／施肥日期，不受分類卡片影響（除非透過分類卡片的批次操作）。

## 刪除植物

刪除植物採「軟刪除」：畫面上會立即消失（含照片），但文字資料仍保留在瀏覽器的 localStorage 中，供之後除錯或資料校對使用，目前沒有復原介面。

## 彙整與趨勢圖

「彙整」卡片顯示目前的總數量／總金額，以及各分類的數量／金額細項。趨勢圖可切換檢視：總數量、各分類數量、總金額（皆為累積趨勢，數量趨勢會反映刪除植物後的實際持有數量變化，金額趨勢則反映累積花費，不因刪除而減少）。

## 設定

標題列 ⚙ 可設定各分類（景天科／仙人球／百合科／天門冬科／鳳梨科／番杏科／大戟科／龍舌蘭／夾竹桃科）的：
- 應澆水天數的**下限**與**上限**（低於下限會提示「才剛澆水沒多久」，高於上限會提示「太久沒澆水了」）
- 應施肥天數的**上限**（高於上限會提示「太久沒施肥了」）

## 本機測試（在電腦上）

```
node serve.js
```

再用瀏覽器開啟 http://localhost:5174

## 安裝到 Android 手機

Service Worker（離線快取＋「加入主畫面」安裝功能）只有在 **https** 或 **localhost** 才會啟用，所以要讓手機真正「安裝」這個 App，需要把這個資料夾放到一個網址上，例如：

1. **GitHub Pages / Netlify / Vercel**：把 `succulent-care-tracker` 資料夾內容部署上去（皆有免費方案），取得一個 https 網址。
2. 用手機 Chrome 開啟該網址。
3. 點右上角「⋮」選單 →「新增至主畫面」或「安裝應用程式」。
4. 之後就會像一般 App 一樣有圖示，可離線使用、資料留在手機本機。

若只是想在同一個 Wi-Fi 下用手機瀏覽器直接使用（不需要真的「安裝」），也可以：
1. 在電腦上執行 `node serve.js`。
2. 查詢電腦的區網 IP（例如 `ipconfig` 看到的 192.168.x.x）。
3. 手機瀏覽器開啟 `http://192.168.x.x:5174`，即可使用（此模式無法安裝為獨立 App，也無離線快取）。

## 檔案結構

- `index.html` / `style.css` / `app.js`：主程式
- `manifest.json` / `sw.js`：PWA 安裝與離線快取設定
- `icons/`：App 圖示
- `serve.js`：本機測試用的簡易伺服器
